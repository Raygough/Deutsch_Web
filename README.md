# 🇩🇪 Deutsch Web
### A German Learning Tool for the Command Line
---
After a year of German, I still struggle with nouns, verbs, and adjectives. Don't even get me started on the many forms of 'the' in German...

I built this tool as a way to practice and catalog one's journey in German from A1 to hopefully, fluency.

Right now this is a command line tool, but I'm working on bringing it to the web so others can learn along with me!

# Technologies used
- React/Typescript
- TailwindCSS
- FastAPI

*to start using:*
`git clone git@github.com:Raygough/Deutsch_Web.git`
